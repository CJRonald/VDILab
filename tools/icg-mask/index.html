🎯 標什麼

  • 只圈 flap（皮瓣/組織）區域 = 解剖邊界，與螢光亮暗無關
  • 暗的皮瓣也要圈進去（灌流好壞是 Stage2 機器算的，不是你標的）
  • 不要只描亮的螢光區（那會錯標成 ICG mask）

🖱 基本操作

  • 點擊模式：點擊逐一加頂點（直邊好用）
  • ✏️ 描邊模式（F 或鈕）：按住拖曳沿輪廓描（彎邊好用）；放開不閉合，可再拖下一段、分次描完
  • 閉合（兩模式共用）：雙擊 或 拖/點回第一頂點（大白點） → 封口成 polygon
  • 描邊與點擊可交替用在同一個 polygon（彎處拖、直處點）；Esc 丟棄未閉合的
  • 所有頂點都可事後選取/拖/刪
  • 點 polygon 內部 → 選取（亮起 + 顯示頂點）
  • 拖頂點 → 調整；右鍵頂點 → 刪除（保留 ≥3 點）
  • 滾輪縮放 · 空白鍵+拖曳 平移

⌨ 快捷鍵

  • click 加頂點 · dblclick 閉合 · F 描邊模式 · Esc 取消繪製中
  • Z 復原 · D/Del 刪選取的 polygon
  • R 旋轉 90° · 微調 ±1° · Enter 存本張
  • G 網格開關 · Shift+G 準星回正中

⊞ 網格 + 準星

  • 方格鎖螢幕恆水平垂直（不跟影像轉）→ 當基準把歪圖轉正
  • 青色十字準星跟游標移動 → 點一下釘住（此點不落頂點）
  • 釘住後移動游標 → 自動解鎖再跟

💾 Export

  • Export JSON (mask)：COCO-style 點座標（原圖座標，schema mask_v1）
  • 訓練端另跑 json_to_mask.py rasterize 成 PNG
  • 標前先填 Rater ID（不同 rater 不互蓋）

🔒 隱私

  • 照片只在你本機處理、永不上傳（純前端）
  • 標註存瀏覽器 localStorage；換機器需自行 Export 帶走
  • Export 自動跳過無 polygon 的空白照片
