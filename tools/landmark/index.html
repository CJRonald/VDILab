使用說明 Esc / 點空白關閉

🖱 基本操作

  • 左側選 landmark → 影像上點一下落點（自動跳下一個未標）
  • 右鍵點已標的點 → 改 visibility / 刪除
  • 滾輪縮放 · 空白鍵+拖曳 平移
  • 點左側清單列可跳到該點；✕ 刪除單點

⌨ 快捷鍵

  • Enter / 💾 鈕 存本張（不跳張，最後一張也能存）· 下一張用 → 鈕 · Z 復原
  • R 旋轉 90° · 微調 ±1° 細轉
  • V 改 active 點 vis · D/Del 刪 active 點
  • G 網格開關 · Shift+G 準星回正中
  • Esc 離開切線模式

⊞ 網格 + 準星

  • 方格鎖螢幕恆水平垂直（不跟影像轉）→ 當基準把歪臉轉正
  • 青色十字準星跟游標移動 → 點一下釘住（此點不落 landmark）
  • 釘住後移動游標 → 自動解鎖再跟

📏 切線模式（變形鼻 NLA）

  • 點切線按鈕 → 在影像上拖一條線貼結構走向（不必找端點）
  • 可重拖覆蓋；按鈕上 ✕ 刪除重畫
  • 右鍵點線 → 改 vis / 刪除 · Esc 離開

💾 Export

  • Export JSON：17 點 + 下巴唇 6 點 + 切線（SimAI 格式，供一致性分析）
  • Export CSV：各角度/比例量測（依 view 自動填欄）
  • 標前先填 Rater ID（不同 rater 不互蓋，供 intra/inter-rater）

🔒 隱私

  • 照片只在你本機處理、永不上傳（純前端）
  • 標註存瀏覽器 localStorage；換機器需自行 Export 帶走
  • Export 自動跳過無標點的空白照片；舊 session 殘留用「🧹 清非本批照片」清掉（避免污染）
