# CGMH Burn Center VDI Lab

林口長庚燒燙傷中心視覺人工智慧實驗室 (Visual-Driven Intelligence Lab) 官方網站

## 關於實驗室

VDI Lab 成立於 2025 年，由林口長庚燒燙傷中心、陽明交通大學與長庚大學人工智慧團隊共同組成，致力於發展結合視覺影像分析、自然語言處理、多模態人工智慧模型與視覺問答技術的臨床應用。

## 研究主題

- **TBSA 與深度 AI 自動辨識**：運用深度學習技術，自動分析燒燙傷面積與深度
- **吸入性嗆傷風險預測模型**：建立預測模型評估患者吸入性嗆傷風險
- **血液製品與手術結果分析**：分析血液製品使用與手術結果的關聯性
